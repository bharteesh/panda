CD Signing Report

Sends a report to the CD Assistant on a bi-weekly basis with new journals that have been signed within the last two weeks and have a status of "rights obtained". This report lists the name of the journal, the licensor, participationDate, collection, discipline, moving wall value, and the ISSN. The report also provides the total number of signed journals, the total number of Archive Collection licensors, the total number of CSP titles, and the total number of publishers with Book agreements.

Owner: CD Assistant (Devin O'Hara)
